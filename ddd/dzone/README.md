# DDD Building blocks

<https://dzone.com/articles/ddd-part-ii-ddd-building-blocks>

- Entities
- Value objects
- Aggregate roots
- Repositories
- Factories
- Services

## Entities

Plain object (model)

- Has identity (ID).
- Can be mutable.
- ID can be used to compare

## Value Objects

Immutable objects.

- No identity.
- comparison based on attributes and types
